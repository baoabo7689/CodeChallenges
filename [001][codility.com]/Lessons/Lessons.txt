Tích 2 số: a*b khi a+b=n (0 < a < b)
Max khi a càng gần n/2

CM:
Xét: a1 = n/2 - x1
=> a1*b1 = (n^2)/4 - x1^2
a2*b2 = (n^2)/4 - x2^2

a2*b2 - a1*b1 = x1^2 - x2^2 
Nếu: x1 > x2 => a1 < a2 < n/2 
	a2*b2 - a1*b1 > 0
Ngược lại: a2*b2 - a1*b1 < 0
=> ĐPCM
